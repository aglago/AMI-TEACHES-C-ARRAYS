ARRAYS

In C/C++, an array is a collection of data items of the same datatype stored at contiguous memory locations.Each data item in an array is called an element and can be reference randomly by indices(later looked at). An example of an array is as follows.


Declaring an array

In declaring an array, use the format data_type array_name[size];


Initializing an array

There are two ways an array can be initialized.
i. Initializing with declaration
ii. Initializing with indices

Initializing with declaration

Once an array is declared, it can be initialized by stating the values of the elements in braces, separating each element with a comma.


In this method of initialization, you can choose to skip defining the size of the array, that way, you can insert as many elements as you can into the array.

Initializing with indices

In this case, the value for each each index is stated separately after the array has been declared. But before that, what are indices? Let us look at how indices work in arrays.

Index Of Elements In Array

The index of an element in an array represents it's position in that array. The first element of an array has index 0 and it increases by one for the subsequent elements. So, the second element of the array is index 1, and the third element, index 2 and it follows.

An index can also be assigned to an element by the programmer, and for this case, the index of the element next to that which an index was assigned to, gets +1 of the index that was assigned, still leaving the first element with index 0.
